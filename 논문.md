## 초록 (Abstract)

퍼스널컬러는 개인의 피부, 모발, 눈동자 색상을 분석해 가장 잘 어울리는 색상 팔레트를 결정하는 방법론으로, 최근 패션 및 뷰티 분야에서 주목받고 있다. 그러나 기존 퍼스널컬러 진단은 전문가에 의한 대면 분석에 의존하며, 진단 이후 실제 의류 선택과의 연결이 부재하다는 한계가 있다.

본 논문에서는 얼굴 사진 분석과 설문 응답을 동적 가중치로 융합해 12계절 퍼스널컬러를 자동 진단하고, 의류 이미지의 semantic segmentation 기반 계절 분류 결과와 결합하여 사용자 맞춤형 코디를 추천하는 통합 시스템을 제안한다.

제안 시스템은 세 가지 핵심 모듈로 구성된다. 퍼스널컬러 진단 모듈은 MediaPipe 얼굴 랜드마크로 15개 관심 영역을 추출하고, Lab 색공간 기반 색상 특징 벡터를 계산해 12계절 시즌 점수를 산출한다. 의류 이미지 분석 모듈은 SegFormer 기반 semantic segmentation으로 의류 부위를 추출하고, 97개 특징과 Random Forest 분류기로 계절 적합성을 예측한다. 추천 모듈은 CIEDE2000 색상 거리, Itten 색상 조화 이론, 날씨 적합성을 다중 기준으로 결합해 코디 조합을 추천한다.

실험 결과, 의류 계절 분류 정확도는 89.74%로 룰 기반 베이스라인(19.2%) 대비 약 4.7배 향상되었다. 사용자 평가에서는 [N]명을 대상으로 추천 만족도를 측정하였으며, [결과 기재 예정]의 긍정적 응답을 얻었다.

**키워드.** 퍼스널컬러, 의류 추천 시스템, semantic segmentation, 색상 조화, 개인화 추천

## 1. 서론

### 1-1. 연구 배경 및 필요성

퍼스널컬러(Personal Color)는 개인의 피부 언더톤, 모발색, 눈동자 색을 종합 분석해 가장 어울리는 색상 계열을 결정하는 색채 분류 체계다 [참고문헌]. 최근 국내외에서 퍼스널컬러에 대한 관심이 급증하고 있으나, 전문가 대면 진단은 비용(평균 5~15만원)과 접근성 측면에서 한계가 있다.

기존의 앱 기반 퍼스널컬러 진단 서비스들은 대부분 간단한 설문만으로 결과를 제공하거나, 이미지 분석을 활용하더라도 진단 이후 실제 의류 선택을 지원하지 않는다는 문제가 있다. 또한 의류 추천 시스템에서 색상 적합성과 날씨 조건을 동시에 고려한 연구는 드물다.

본 연구에서는 다음 세 가지 문제를 해결하는 통합 시스템을 제안한다.

1. **정확한 자동 퍼스널컬러 진단.** 얼굴 사진과 설문을 동적 가중치로 융합해 12계절 진단 정확도를 높인다.
2. **의류 이미지의 계절 적합성 자동 분류.** SegFormer 기반 segmentation과 Random Forest를 결합해 의류 이미지에서 계절 정보를 추출한다.
3. **다기준 코디 추천.** 퍼스널컬러 적합도, 날씨 조건, 색상 조화를 통합한 점수 체계로 코디를 추천한다.

### 1-2. 연구 목적 및 기여

본 논문의 주요 기여는 다음과 같다.

- 얼굴 이미지에서 Lab 색공간 기반 4축 특징 벡터(색온도/명도/선명도/대비)를 추출하고, 사진 품질에 따른 동적 가중치 융합 기법을 제안한다.
- SegFormer semantic segmentation 결과와 97개 특징 기반 Random Forest를 결합한 의류 계절 분류 파이프라인을 구현하고, 89.74%의 분류 정확도를 달성한다.
- CIEDE2000 색상 거리와 Itten 색상 조화 이론을 결합한 다기준 코디 추천 알고리즘을 제안한다.
- 이 세 모듈을 하나의 웹 애플리케이션으로 통합하고 실제 사용 가능한 시스템을 구현한다.

### 1-3. 논문 구성

본 논문의 구성은 다음과 같다. 2장에서 퍼스널컬러 분석, 의류 이미지 인식, 패션 추천 시스템 관련 선행 연구를 검토한다. 3장에서 제안 시스템의 전체 설계를 설명한다. 4장에서 세 핵심 모듈의 구현 방법을 상세히 기술한다. 5장에서 실험 결과를 제시하고 분석한다. 6장에서 결론과 향후 연구 방향을 제시한다.

## 2. 관련 연구

### 2-1. 퍼스널컬러 분석 시스템

퍼스널컬러의 색채 이론적 기반은 Itten(1961)의 색상환과 Munsell 색채 체계에서 출발했으며, 이후 12계절 분류 체계로 발전하였다 [참고문헌]. 컴퓨터 비전을 활용한 자동 퍼스널컬러 진단 연구로는 XXX 등(연도)이 RGB 색공간에서 피부색을 분류한 연구가 있으나, 조명 변화에 취약하다는 한계를 보였다. 본 연구에서는 Lab 색공간의 b* 축을 색온도 측정에 사용함으로써 이 문제를 완화한다.

### 2-2. 의류 이미지 인식 및 분류

의류 이미지 segmentation 분야에서는 Deeplab, Mask R-CNN 등이 활용되어 왔다. 최근 Transformer 기반의 SegFormer(Xie et al., 2021)는 다양한 해상도의 특징을 효율적으로 활용하며 의류 도메인에서 우수한 성능을 보인다. Jiang et al.(연도)은 패션 이미지에서 속성을 분류하는 연구를 진행했으나, 의류의 날씨 적합성 예측을 직접 다룬 연구는 드물다.

의류 속성 분류에 Random Forest를 활용한 연구로는 XXX 등(연도)이 있으며, 색상, 질감, 형태 특징을 결합했을 때 단일 모달리티 대비 성능이 향상됨을 보였다. 본 연구는 segmentation 결과에서 추출한 97개 이종 특징을 Random Forest 입력으로 사용하는 파이프라인을 제안한다.

### 2-3. 패션 추천 시스템

패션 추천 시스템은 협업 필터링 기반, 콘텐츠 기반, 하이브리드 방식으로 분류된다. He and McAuley(2016)는 사용자 구매 이력과 시각적 특징을 결합한 추천 모델을 제안했다. 색상 조화를 고려한 추천 연구로는 XXX 등(연도)이 Itten 색상 이론을 패션에 적용한 사례가 있다.

기존 연구들과 달리 본 연구에서는 퍼스널컬러 적합도, 날씨 조건, 색상 조화를 동시에 고려하는 다기준 점수 체계를 제안하며, 별도의 사용자 구매 이력 데이터 없이 퍼스널컬러 진단 결과만으로 개인화 추천을 실현한다.

## 3. 시스템 설계

### 3-1. 전체 아키텍처

제안 시스템은 세 모듈과 프론트엔드로 구성된다.

```
[사용자]
  ├─ 얼굴 사진 + 설문
  │    ↓
  │  [모듈 1] 퍼스널컬러 진단
  │    → 12계절 결과 + 24색 팔레트
  │
  ├─ 의류 사진 업로드
  │    ↓
  │  [모듈 2] 의류 이미지 분석
  │    → 계절 태그 + 대표색
  │
  └─ 추천 요청
       ↓
     [모듈 3] 코디 추천 알고리즘
       → 퍼스널컬러 38% + 날씨 22% + 조화 28% + 안정성 12%
       → 상위 60개 코디 조합
```

**프론트엔드.** React + TypeScript SPA. 7개 페이지(진단/옷장/추천/저장/가상착용 등).

**백엔드.** FastAPI 서버. 배경 제거(`/api/background/remove`)와 의류 정밀 추출(`/api/clothing/extract`) 두 엔드포인트.

**저장소.** 사용자 데이터는 브라우저 localStorage에 저장. 서버 DB 없음 (데모 단계 설계).

### 3-2. 데이터 구조

**의류 아이템 (`ClothingItem`).** 카테고리(상의/하의/아우터/신발/액세서리), 날씨 계절 태그, 대표 HEX, 3색 우세색 배열, 퍼스널컬러 진단 이전 점수 등을 포함한다.

**퍼스널컬러 결과 (`FinalResult`).** 12계절 중 Top1/Top2 시즌, 24색 팔레트, 신뢰도(0~1), 사진/설문 융합 가중치를 포함한다.

**카탈로그.** 860개 의류 이미지와 메타데이터(TypeScript 상수). SegFormer 파이프라인으로 계절 분류 및 색상 추출 완료.

## 4. 구현

### 4-1. 퍼스널컬러 진단 모듈

#### 4-1-1. 얼굴 색상 추출

MediaPipe FaceLandmarker를 사용해 얼굴 468개 랜드마크를 검출하고, 이를 기반으로 15개 관심 영역(ROI)을 정의한다. ROI에는 양쪽 볼, 이마, 코, 눈 아래, 턱선(피부), 홍채(눈), 눈썹, 입술, 헤어라인이 포함된다.

각 ROI에서 픽셀을 추출할 때는 이상치를 제거하기 위해 피부 영역은 Lab medoid 방식을, 입술 영역은 붉은/분홍 색조 픽셀 우선 방식을 적용한다. 조명 보정은 흰 종이 기준 → 배경 중립 픽셀 → 코너 패치의 3단계 fallback으로 처리한다.

피부 대표색 가중치: 볼 40%, 눈 아래 20%, 코 16%, 이마 14%, 턱선 10%

#### 4-1-2. 색상 특징 벡터 계산

추출된 skin/hair/eyes/lips 색상에서 4축 특징 벡터를 계산한다.

**색온도 (temperature).** RGB 채널 차이 방식은 조명 색온도에 민감하다는 문제가 있다. 본 연구에서는 Lab 색공간의 b* 축(음수: 파랑/쿨, 양수: 노랑/웜)을 활용해 조명 변화에 robust한 색온도 지수를 계산한다.

```
temperature = labTemperatureIndex(skin)×0.45 + labTemperatureIndex(lips)×0.25
            + labTemperatureIndex(hair)×0.15 + labTemperatureIndex(eyes)×0.15
```

**대비 (contrast).** 동아시아 사용자에서 검은 머리카락만으로 겨울 고대비로 과분류되는 문제를 방지하기 위해 머리카락의 대비 기여를 22%로 제한한다.

```
contrast = facialContrast(skin, eyes, lips)×0.78 + hairContrast(skin, hair)×0.22
```

#### 4-1-3. 12계절 점수 산출

각 12계절에 대해 팔레트 점수와 특징 유사도 점수를 계산해 합산한다.

**팔레트 점수.** 추출된 피부/모발/눈/입술 색상과 해당 시즌 24색 팔레트 간 CIE76 색차 최솟값을 0~1로 변환한다.

**특징 유사도.** 추출된 4축 특징 벡터와 각 시즌의 기준 특징 벡터(traits) 간 유사도를 가중합산한다. 선명도 비중(0.34)이 가장 높다.

**도메인 보정.** 저채도 얼굴이 겨울 시즌으로 과분류되는 것을 방지하는 페널티, soft 시즌에 대한 보너스 등 규칙 기반 보정을 추가한다.

#### 4-1-4. 사진-설문 동적 융합

사진 품질 점수(0~1)에 따라 사진과 설문의 가중치를 동적으로 조정한다.

```
photoWeight = clamp(0.22 + photoQuality × 0.14, 0.22, 0.36)
```

사진 품질이 낮을수록 설문 비중이 높아져 조명 불량 환경에서도 안정적인 결과를 제공한다. 설문은 5문항(손목 혈관색, 액세서리 금속, 컬러 선호도, 대비 스타일, 색 깊이 선호)으로 구성하며, 각 선택지는 4축에 가중치를 갖는다.

### 4-2. 의류 이미지 분석 모듈

#### 4-2-1. 의류 부위 추출 — SegFormer

HuggingFace의 `sayeed99/segformer-b3-fashion` 모델을 사용해 입력 이미지에서 픽셀별 의류 라벨(39개 카테고리)을 예측한다. 목표 부위(상의/하의/아우터/신발 등)에 해당하는 라벨 ID의 픽셀만 마스크로 추출하고, morphological 처리(close/open, hole fill, 상위 3개 연결 성분 유지)로 노이즈를 제거한다.

#### 4-2-2. 의류 계절 분류 파이프라인

**라벨 및 색상 추출.** 마스크 영역에서 SegFormer 예측 라벨과 KMeans(최적 k: silhouette score로 자동 선택) 기반 우세색을 추출한다.

**룰 기반 예측 (베이스라인 및 특징).** 라벨별 계절 점수 테이블, 소매 비율(sleeve_ratio), 의류 노출 비율(exposure), 색상 밝기 기반 룰로 초기 계절을 예측한다. 이 결과는 독립 베이스라인이면서 ML 모델의 입력 특징으로도 사용된다.

**97개 특징 추출.** 라벨 픽셀 비율(34개), 바운딩박스(6개), 형태(2개), 색상 통계(13개), 품질(8개), fine/detail 라벨 원핫, 우세색 5개 RGB·비율, 룰 점수·신뢰도·예측 원핫을 포함한다.

**Random Forest 분류.** 300개 트리, balanced class weights로 학습. 학습 데이터는 직접 라벨링한 516개 의류 이미지.

**재질 예측.** fine_label과 매핑 테이블로 결정 (sweater→knit, coat→wool, jacket→nylon 등).

### 4-3. 추천 알고리즘 모듈

#### 4-3-1. 퍼스널컬러 적합도 점수

의류 `dominantColors` 상위 3색에 대해 비율 가중 평균으로 퍼스널컬러 적합도를 계산한다.

단일 색상 점수 산출 시 CIEDE2000 색차를 사용한다. CIE76 대비 CIEDE2000은 파란 계열에서 인간 지각 상관계수가 0.176 → 0.289로 향상된다 (PubMed 기준). 쿨 퍼스널컬러 팔레트 매칭에서 정확도가 높아 채택했다.

```
paletteScore = max(0, 100 - minDeltaE2000(item, palette) × 4.5)
avoidPenalty = minDeltaE2000(item, avoidColors) < 10 ? 22 : < 16 ? 10 : 0
utilityBonus = isNeutral || isDenim ? 8 : 0
```

등급: BEST ≥ 88 / GOOD ≥ 74 / OK ≥ 58 / CHECK < 58

#### 4-3-2. 날씨 적합성 점수

날씨를 8개 구간(4도 이하 ~ 28도 이상)으로 분류하고, 각 구간별 허용 키워드와 의류 seasonTag를 매칭한다. 9도 이하에서 여름 의류, 20도 이상에서 겨울 의류를 필터링한다.

#### 4-3-3. 색상 조화 점수 — Itten 이론

두 의류의 대표색 hue 각도 차이를 기반으로 Itten 색상 이론 5가지 조화 타입을 분류하고 점수를 부여한다.

| 조화 타입 | hue 각도 차 | 특징 |
|----------|------------|------|
| 단색 조화 | ≤ 15° | 자연스럽고 세련 |
| 유사색 | ≤ 45° | 편안한 조화 |
| 긴장감 | 30~120° | 역동적 |
| 삼각 조화 | ≈ 120° | 균형 있는 대비 |
| 보색 | ≥ 150° | 강한 대비 |

#### 4-3-4. 코디 조합 생성 및 종합 점수

상의 × 하의 × (아우터 또는 없음)의 조합을 전수 생성하고, 아래 가중 합산으로 종합 점수를 계산해 상위 60개를 반환한다.

```
종합점수 = 퍼스널컬러 × 0.38 + 날씨 × 0.22 + 조화 × 0.28 + 안정성 × 0.12
```

동일 아이템이 추천을 독점하지 않도록 아이템별 최대 3회 등장 제한(`diversifyRecommendations`)을 적용한다.

## 6. 결론 및 향후 과제

### 6-1. 결론

본 논문에서는 퍼스널컬러 진단, 의류 이미지 분석, 다기준 코디 추천을 통합한 AI 기반 의류 추천 시스템을 제안하고 구현하였다.

주요 기여는 다음과 같다.

첫째, 얼굴 이미지에서 Lab 색공간 기반 색온도 특징을 추출하고, 사진 품질 반응형 동적 가중치 융합으로 조명 조건에 robust한 12계절 퍼스널컬러 진단 방법을 제안하였다.

둘째, SegFormer semantic segmentation과 97개 이종 특징 기반 Random Forest 분류기를 결합해 의류 이미지에서 계절 적합성을 89.74% 정확도로 예측하는 파이프라인을 구현하였다.

셋째, CIEDE2000 색상 거리와 Itten 색상 조화 이론을 결합한 다기준 추천 알고리즘을 제안하였으며, 퍼스널컬러 적합도·날씨 조건·색상 조화·보유 안정성의 4축 가중 점수 체계로 개인화된 코디를 추천한다.

### 6-2. 향후 과제

- **의류 자동 등록.** 사용자가 자신의 옷 사진을 찍으면 계절 분류·색상 추출·카테고리 감지가 자동으로 이루어져 옷장에 등록되는 엔드투엔드 파이프라인 구현.
- **코디 평가 기능.** 사용자가 직접 선택한 코디 조합에 대해 퍼스널컬러 적합도와 색상 조화 점수를 실시간으로 피드백하는 기능.
- **human_label 데이터 확장.** 현재 402개 미완료 라벨을 완성하고, 추가 데이터 수집으로 모델 성능 향상.
- **사용자 피드백 반영.** 착용 이력과 선호도를 반영한 협업 필터링 도입.
- **모델 경량화.** 모바일 브라우저 내 추론 가능한 경량 분류 모델 탐구.
